เวลาสร้างหน้าใหม่ให้ สร้างคลาสใหม่ซึ่ง extends คลาส Page

โดยการเขียน GUI ให้ทำใน Constructor เหมือนที่จารย์แบงค์ให้ทำใน Lab
ในคลาส Page มี attribute ชื่อ mainPanel
ให้ใส่ UI ทุกอย่างลงใน MainPanel แล้ว setVisible เป็น true
